## Short English Post

I built CodexVisual, a small open-source tool for people using Codex heavily.

It shows the remaining Codex 5-hour and 7-day quota directly in the macOS menu bar or Windows taskbar tray. It reads local Codex logs, ignores expired events, and does not read auth tokens or call an external quota API.

macOS and Windows downloads are available here:
https://github.com/orangeshushu/CodexVisual

## Longer Product Description

CodexVisual is a focused desktop utility for developers who use Codex and want quota visibility without opening logs. It displays the remaining 5-hour quota and 7-day quota in a compact macOS menu bar item or Windows tray dashboard. The app reads local Codex session/log files, filters expired quota events, supports manual and smart refresh, and provides English and Chinese UI.
